# ICMP fields [exported-fields-icmp]

None


## icmp [_icmp]

IP ping fields.


**`icmp.requests`**
:   Number if ICMP EchoRequests send.

type: integer


## rtt [_rtt]

ICMP Echo Request and Reply round trip time


**`icmp.rtt.us`**
:   Duration in microseconds

type: long
